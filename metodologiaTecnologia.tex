\chapter{Tecnologías y Metodologías}
\section{Metodología}
Para el desarrollo del proyecto se ha seguido una metodología \gls{iterinc}, permitiendo
la adaptación a cambios y la incorporación de nuevas funcionalidades de manera gradual.
Originalmente se planteó en 4 incrementos principales:
\begin{enumerate}
    \item API Backend y Modelado de Datos.
    \item Comunicación Backend–Frontend y Modelación de la API.
    \item Generación Dinámica de Formularios (Formly).
    \item Generación Automática de Esquemas desde la Base de Datos.
\end{enumerate}
